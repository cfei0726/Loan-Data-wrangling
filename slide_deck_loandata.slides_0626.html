Exploratory Analysis of Prosper Loan Data¶

by Chen Yi Fei¶

Investigation Overview¶

The presentation is to show observation of main features that explain Lender Yield cost and defaulted cases.

Dataset Overview¶

This dataset is financial dataset and this is related to the loan, borrowers, lenders, interest rates and stuffs like that. This data set contains 113,937 loans with 81 variables on each loan, including loan amount, borrower rate (or interest rate), current loan status, borrower income, and many others
